سیستم جامع فروشگاه آنلاین و مدیریت کیف پول دیجیتال

این پروژه یک سیستم فروشگاهی کامل و مقیاس‌پذیر است که علاوه بر قابلیت‌های خرید و سفارش، دارای یک سیستم کیف پول داخلی پیشرفته و مدیریت دسترسی‌های سطح بالا می‌باشد.
🚀 ویژگی‌های کلیدی
۱. بخش فروشگاهی

    مدیریت محصولات: دسته‌بندی دقیق محصولات برای ناوبری آسان.
    جستجوی هوشمند: قابلیت جستجوی سریع در میان کاتالوگ محصولات.
    سبد خرید و سفارش: فرآیند آسان اضافه کردن کالا به سبد خرید ثبت سفارش و پرداخت نهایی.

۲. سیستم کیف پول و پرداخت

    کیف پول دیجیتال: امکان شارژ موجودی کیف پول و پرداخت مبالغ خرید از این طریق.
    انتقال وجه: امکان واریز موجودی از کیف پول یک کاربر به کیف پول کاربر دیگر.
    پرداخت کارت به کارت: ادغام سیستم پرداخت کارت به کارت برای خریدهای مستقیم.

۳. امنیت و دسترسی‌ها

    احراز هویت (Authentication): استفاده از استاندارد JWT (JSON Web Token) برای امنیت حداکثری و مدیریت نشست‌های کاربران.

    سطوح دسترسی (RBAC): تفکیک کامل سطوح دسترسی بین کاربر عادی و ادمین سیستم.

    ** امنیت تراکنشها ( امنیت برای اینکه دوبار تراکنش انجام نشود با حالت خوشبینانه ورژن و ایدمپوتنسی یکتا

۴. پنل مدیریت (Admin Dashboard)

    **مدیریت دسترسی‌هاادمین سیستم این اختیار را دارد که قوانینی را برای کاربران وضع کند . قوانین کاملا پویا هستند میتواند برای کاربران دارای سن و کد ملی و ستونهای توضیحی کاربران قوانینی وضع کند برای مقدار مجاز کارت به کارت و پرداخت (مثلاً محدود کردن خرید یا مسدود کردن قابلیت‌های تراکنش برای کاربران خاص).
